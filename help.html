Using Serial Communication

The first thing you see when Cereal Com starts is the setup menu. Here you can change settings and configure the serial port before you connect. Once you are finished setting up you have 2 options:

  1. Close the window using the X at the top or
  2. Open the serial port (this also closes the setup window)
If you decided to click the 'Open' button the settings window closed and an attempt was made to open the serial port using the settings you chose. If the serial port opened successfully then you can start sending an receiving data through the port. All you have to do is type with the window selected to send characters. Not all characters have been tested but all letters, numbers, punctuation keys and the 'Esc' key all work correctly. Control characters have not been tested. If you are having trouble sending a specific character e-mail me and I will try to add it.

You will see 3 buttons are available to you:
  1. Stop - This button will disconnect the serial port
  2. Save - This button will record data received on the serial port
  3. Send - This button will send a file to the serial port of your choosing
If you decided to close the settings window using the X at the top the window closed and the serial port did not open. You can do a few things here:
  1. Open - This button opens the serial port using the settings you chose previously
  2. Settings - This button opens the settings menu again
  3. Help - This button opens the help window

Saving Data

When you have continuous data coming to the serial port that you have open you can store that data in a text file on your computer.

  1. Open the serial port
  2. Click the Save button
  3. Choose where you want to save the file and what you want to name it
  4. Start storing data
Cereal Com will continue to store data until you click the Stop button.

Sending Files

To send a file you must follow these steps:

  1. Open the serial port
  2. Click the Send button
  3. Choose the file you want to send
  4. Wait while the file is sent
Only text files have been tested. If you want to see the progress of the file being sent choose to echo characters in the settings window.

FAQs

Why do I keep getting an error message when I try to open the serial port?
You probably already have that port open in another program. There are other Chrome apps that use serial ports that do not properly close the serial connection when the window closes. Unfortunately I cannot help you fix that. If you are trying to use a bluetooth port or other type of port I'm not sure. No testing has been done on anything other than serial ports.

What types of files can I send through the serial port?
Good question. Only text files have been tested. I'm not sure what other kinds of files you would want to send over such a slow connection.

What is the echo function for?
By default any character you type will not appear in the window. If you want or need to see what you are typing then check the box for echo.

What are CR and LF?
CR stands for Carriage Return and LF stands for Line Feed. You can think of it in terms of a typewriter. CR will send you back to the beginning of a line. LF will start a new line. The data could show up in an unusual way if you do not have both. For example, if you only have CR coming from your device and no LF you might see the same line be written over and over instead of seeing every string on a new line.

What is the character delay for?
Depending on your computer you may run into issues with characters being sent to quickly. This is mainly for sending text files. If the characters send too quickly your connected device may miss a few and produce undesirable results. Increasing the character delay should resolve that issue.

Are the settings stored so I don't have to choose them again at every startup?
Yes. Settings should sync to your Chrome login so settings carry over from one computer to another. The settings are saved every time you open a connection to a serial port.

Does Cereal Com close the serial port when I close the app?
Yes. Cereal Com closes the port properly when you close the app window or when you click the stop button. Most of the other serial communication apps in the Chrome Web Store do not.

What are the permissions for?
Access to Serial Ports is for connecting to the serial port obviously. Writing to files on your system is for 2 things. First it is for storing settings so you don't have to choose your settings every time you open the app. Second it is for collecting data from the serial port and storing in a file on your computer. This app does not store any personal information or anything other than the settings and whatever you choose to store from the serial port.

Does Cereal Com collect data for analytics?
No. There is nothing in Cereal Com that collects information on you, how you use the app or anything else. I don't care how you use the app, I just want you to use it and benefit from it.

How does Binary work?
Some devices send information via binary (1s and 0s). Checking the 'Convert Binary to Hex' box will take in that binary data and turn it into readable Hex values. Every byte of binary data can be converted to Hex. Instead of seeing random garbage characters, you will now see numbers 0-9 and letters A-F in pairs (for example: AB or 3F). You can also choose how often to put the incoming binary on a new line using the 'Binary characters before a new line' box. This is useful if you know you have a certain number of bytes coming in.
